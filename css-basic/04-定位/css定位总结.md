**属性值**

| 键       | 描述                                          |
| -------- | --------------------------------------------- |
| static   | 自动定位                                      |
| relative | 相对定位,相对于上一个已经定位的父元素进行定位 |
| absolute | 绝对定位,相对于上一个已经定位的父元素进行定位 |
| fixed    | 固定定位,相对于浏览器窗口进行定位             |

指定定位在哪里,需要通过偏移值来指定

| 键     | 描述                     |
| ------ | ------------------------ |
| top    | 相对于父元素上边线的距离 |
| bottom | 相对于父元素下边线的距离 |
| left   | 相对于父元素左边线的距离 |
| right  | 相对于父元素右边线的距离 |

**静态定位(static)**

在静态定位的状态下,top,right,bottom,left,z-index属性无效

**相对定位**

- 参照元素原来的位置进行移动
- 通过left,right,top,bottom属性进行定位
- 元素原有的空间进行保留
- 元素在移动时会盖住其他元素
